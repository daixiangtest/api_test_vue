# 技术栈说明
技术栈：vue3 + elemntPlus + axios + vue-router + pinia + scss开发
# 目录介绍
        api目录：封装后端交互的接口请求

        assets目录：存放文件和js交互的文件

        router目录：路由配置

        stores: pinia全局状态管理目录

        style: 全局的样式配置文件

        utils目录： 封装的全局工具文件

        Views：项目的视图文件存放目录

        App.vue:项目根组件

        main.js：项目js入口文件
